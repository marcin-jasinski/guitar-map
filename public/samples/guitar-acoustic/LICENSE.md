# Guitar-acoustic samples — licence & attribution

These `.mp3` files are a subset of the **guitar-acoustic** sample set from
[nbrosowsky/tonejs-instruments](https://github.com/nbrosowsky/tonejs-instruments),
which sources them from the **Versilian Community Sample Library (VCSL)**.

- **Samples:** released under **[Creative Commons Attribution 3.0 (CC-BY 3.0)](https://creativecommons.org/licenses/by/3.0/)**.
  Redistribution is permitted with attribution — hence this file, which ships alongside the audio.
- **Attribution:** VCSL (Versilian Studios) via the tonejs-instruments project
  (© 2018 Nicholaus P. Brosowsky, project code under MIT).

Only these eight notes are committed — D2, G2, C3, F3, A3, D4, G4, D5 — one every
few semitones. `Tone.Sampler` pitch-shifts them across the whole range, so the set
stays small (a few notes per octave, not one per fret).
